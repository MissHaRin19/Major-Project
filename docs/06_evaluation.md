# Evaluation

## Dataset

- MIT-BIH ECG dataset used for normal data
- Synthetic attack data generated through simulation

## Metrics Used

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

## Evaluation Process

- Normal and attack data combined
- Models evaluated on classification performance
- Reconstruction error used for anomaly detection (LSTM)

This approach ensures both detection capability and reliability are measured.
