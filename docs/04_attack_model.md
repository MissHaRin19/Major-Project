# Attack Simulation

The system simulates four major cyberattack types:

## Data Injection
Malicious values are inserted into telemetry data, causing abnormal readings.

## Replay Attack
Previously captured valid data is resent, misleading the system with outdated information.

## Man-in-the-Middle (MITM)
Data is intercepted and modified during transmission, altering physiological signals.

## Denial of Service (DoS)
Interrupts or delays communication, preventing timely data updates.

## Impact

These attacks can:
- Distort ECG signals
- Misrepresent patient condition
- Delay or block critical data

The simulation allows safe testing of these attack scenarios.
