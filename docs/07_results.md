# Results

## LSTM Autoencoder Performance

- Accuracy: ~0.73
- Precision: ~0.94
- Recall: ~0.50
- F1 Score: ~0.65

## Isolation Forest Performance

- Accuracy: ~0.57
- Precision: ~0.70
- Recall: ~0.23
- F1 Score: ~0.35

## Observations

- LSTM shows high precision → reliable detection
- Recall is moderate → some attacks missed
- Isolation Forest performs significantly worse

## Conclusion

LSTM Autoencoder is more effective for detecting anomalies in time-series biomedical data compared to traditional methods.
