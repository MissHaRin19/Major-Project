# Limitations

- Recall is moderate (~0.50), meaning some attacks are not detected
- Simulation does not fully replicate real-world pacemaker hardware
- Synthetic data may not capture all real physiological variations
- Limited dataset size affects model generalization

## Future Improvements

- Improve recall through threshold tuning
- Use larger and more diverse datasets
- Integrate advanced models (e.g., Transformer-based models)
- Extend system to support multiple medical devices
